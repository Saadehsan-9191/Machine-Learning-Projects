Bank Marketing Classification Project
Predicting Term Deposit Subscription Using Machine Learning

Project Overview
This project uses the Bank Marketing Dataset from the UCI Machine Learning Repository to build and compare several machine learning classification models. The goal is to predict whether a customer will subscribe to a term deposit after a marketing campaign. This is a real-world business problem used by banks to improve marketing efficiency, reduce costs, and target the right customers.

Dataset Information
Source: UCI Machine Learning Repository
Dataset Name: Bank Marketing
Number of Rows: 45,211
Number of Features: 17
Target Variable: y (yes or no, whether the client subscribed to a term deposit)

Feature Categories
Client attributes: age, job, marital status, education, default, housing loan, personal loan
Campaign attributes: contact type, month, day, duration, number of contacts
Historical campaign performance: previous outcome, number of previous contacts
Economic indicators: employment variation rate, consumer confidence index, etc.
